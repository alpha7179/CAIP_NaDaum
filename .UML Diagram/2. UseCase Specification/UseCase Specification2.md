## **UC-02: UC-02 종합 위험도 평가 및 안전망 연결**

| 항목 | 내용 |
| --- | --- |
| **Use Case ID** | **UC-02** |
| **Use Case 명** | **종합 위험도 평가 및 안전망 연결** |
| **설명** | **UC-01 대화 종료 시점 또는 대화 중 실시간 고위험 신호 감지 시점에, 시스템이 텍스트 감정 수치·음성 프로소디·고위험 키워드를 통합 분석하여 위험도(저/중/고)를 판단하고, 단계별 안전 프로토콜에 따라 자동으로 적절한 개입을 수행한다.** |
| **Primary Actor** | **일반 사용자 (청년/청소년)** |
| **Secondary Actor** | **외부 전문 상담 기관, 보호자** |
| Precondition | **UC-01 대화 세션이 종료된 상태이거나, 대화 중 실시간 고위험 신호가 감지된 상태** |
| Postcondition | **위험도 레벨에 따른 개입이 완료되고, 위기 이벤트 로그가 DB에 적재된다. 중·고위험의 경우 안부 알림 스케줄이 등록되며, 고위험의 경우 보호자 알림 및 전문 기관 연결이 시도** |

### 기본 성공 시나리오 (Main Flow)

| 단계 | 행위자 | 내용 |
| :-: | :--: | :-- |
| 1 | System | UC-01 종료 또는 실시간 고위험 신호 감지를 트리거로 본 유스케이스가 자동 실행된다 |
| 2 | System | 텍스트 감정 수치(1~10 척도), 음성 프로소디 지표, 고위험 키워드 탐지 결과를 통합한다 — **"감정 수치 통합"** |
| 3 | System | **KoELECTRA 자살위험 탐지 모델** 로 종합 위험도 평가를 수행한다 — **"종합 위험도 평가 수행"** |
| 4 | System | 위험도 판단 결과에 따라 분기한다 (저위험 / 중위험 / 고위험 중 하나) |
| **5H-1** | System | **[고위험 분기]** **두 안전 조치를 동시 시도**한다 (Fork):<br>① **전문 심리 상담 기관 연결 시도** (UI 표시)<br>② **보호자 알림 발송 시도** `<<extend>>` 상담 연결 / 보호자 연결 |
| **5H-2** | System · User | ①-1**[기관 연결 승인 시]** 기관 연결을 진행한다<br>①-2**[기관 연결 거부 시]** 긴급 전화번호(정신건강 위기상담 전화 1577-0199, 자살예방상담 1393)를 안내하고 연결한다 |
| **5H-3** | System | ②-1**[보호자 연락처 등록시]** 보호자에게 알림을 즉시 발송한다<br>②-2**[보호자 연락처 미등록시]** 사용자에게 보호자 등록을 요청하는 팝업을 표시한다 |
| 6 | System | (고위험 한정) 위 5H-1~5H-3 단계가 모두 완료되면 다음 단계로 진행한다 (Join) |
| 7 | System | (중·고위험 한정) **지속 모니터링 필요 판단**을 수행한다 |
| 8 | System | (중·고위험 한정) **24시간 후 자동 안부 확인 메시지 발송 스케줄을 등록**한다 `<<extend>>` 안부 알림 및 재대화 유도 |
| 9 | System | (저/중/고위험 공통) 모든 위험도 이벤트를 **이벤트 로그 DB에 적재**한다 |
| 10 | System | **지속적 기록을 요청**하고 홈화면 복귀를 안내한다 |
| 11 | User | 일기 확인 및 홈화면으로 복귀한다 |

### 대안 시나리오 (Alternative Flow)

#### 1a. 대화 중 실시간 고위험 키워드 감지 (UC-01에서 트리거)
| 단계 | 행위자 | 내용 |
| :-: | :--: | :-- |
| 1a.1 | System | UC-01 대화 루프 중 자살·자해 키워드 또는 임계치 초과 감정 지표가 감지되면 대화를 즉시 중단한다 |
| 1a.2 | System | 본 UC의 단계 5부터 즉시 실행한다 (위험도 평가는 이미 트리거 시점에 고위험으로 확정) |

#### 5L/5M. 저위험/중위험 분기 시
| 단계 | 행위자 | 내용 |
| :-: | :--: | :-- |
| **5L** | System | **[저위험 분기]** 일상 관리 팁을 제공한다 → **사용자가 일상 관리 팁을 확인**한다 → 단계 9로 진행 `<<extend>>` 일상 팁 제공 |
| **5M** | System | **[중위험 분기]** 상담 서비스를 추천한다 → **사용자가 상담 서비스 추천을 확인**한다 → 단계 7로 진행 `<<extend>>` 상담 서비스 추천 |

#### 7a. 사용자가 24시간 후 안부 알림에 무응답 또는 비활성
| 단계 | 행위자 | 내용 |
| :-: | :--: | :-- |
| 7a.1 | System | 사용자가 안부 알림에 응답하지 않을 경우 추가 안내 메시지를 발송한다 |
| 7a.2 | System | 지속적으로 응답이 없으면 보호자에게 후속 알림을 발송한다 (고위험 대상자 한정) |


### 관련 유스케이스
- `<<extend>>` 일상 팁 제공 (저위험 시)
- `<<extend>>` 상담 서비스 추천 (중위험 시)
- `<<extend>>` 상담 연결 (고위험 시)
- `<<extend>>` 보호자 연결 (고위험 시)
- `<<extend>>` 안부 알림 및 재대화 유도 (중·고위험 시)
